# open-gitmoji

## Features
- [x] Keywords emoji tips and input（Emoji 提示及插入）.
- [x] Language config（语言配置）.
- [x] custom prompt typography（自定义提示排版）.
- [x] start with `:`（`:`开头才提示配置）
